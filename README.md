# GreenData Authorization v0.1

Проверка страницы авторизации.
Сценарии включают в себя проверку авторизации с верными данными. Проверка с неверными данными. 
Проверка авторизации со включенным флагом "Запомнить меня"(Назначение чекбокса отличается то привычного, по хорошему надо доработать). 
Проверка авторизаци с доменной УЗ. Проверка авторизации с текущей УЗ.

Сценарии были создани с использованием стека Java + Selenide + TestNG. Для отчетности использован Allure.

Запуск сценариев проводится следующим образом: 
- Для запуска из IDE необходимо найти src/test/resources/testNg.xml, запусаем его
- Для запуска из командной строки\CI - передать mvn clean test
